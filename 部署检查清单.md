## 📝 给其他开发者的说明

### 首次克隆项目后

1. **阅读启动指南**
   - 中文：`启动指南.md`
   - 英文：`STARTUP_GUIDE.md`

2. **检查前置要求**
   - Docker Desktop 已安装并运行
   - .NET SDK 10.0 已安装
   - Flutter SDK 已安装
   - Android Studio 已安装（或使用真机）

3. **按照启动指南操作**
   - 严格按照顺序：数据库 → 后端 → 前端
   - 每个步骤都要验证是否成功

4. **遇到问题**
   - 查看启动指南的"常见问题排查"部分
   - 检查各服务的日志输出
   - 验证网络和防火墙设置
