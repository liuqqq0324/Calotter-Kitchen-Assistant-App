# Python项目

这是一个Python项目模板。

## 项目结构

```
.
├── main.py           # 主程序入口
├── requirements.txt  # 项目依赖
├── .gitignore       # Git忽略文件
└── README.md        # 项目说明
```

## 安装依赖

```bash
pip install -r requirements.txt
```

## 运行项目

```bash
python main.py
```

## 开发环境

建议使用虚拟环境：

```bash
# 创建虚拟环境
python -m venv venv

# 激活虚拟环境
# macOS/Linux:
source venv/bin/activate
# Windows:
# venv\Scripts\activate

# 安装依赖
pip install -r requirements.txt
```
